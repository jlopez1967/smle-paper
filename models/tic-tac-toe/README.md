# Tic-Tac-Toe Exactor Model

This directory contains the literal 186-rule segment model discovered by Symbolic Machine Learning via Exactor (SMLE) that achieves 99.8% precision at 99.8% recall on the **Tic-Tac-Toe Endgame** dataset.

This is the exact model used for the validation results highlighted in the main paper.

### Files

- `exactor_model.py`: The stand-alone Python script that evaluates the 186 First-Order/Propositional Boolean rules discovered by the engine against the 9-feature Tic-Tac-Toe board inputs. It includes the `MicroMindClassifier` compatible with Scikit-Learn.
- `The_Victory_Genome.pdf`: A visual analysis report generated by the engine that explains the rule distribution and optimization process.

### How to Use

The Python script `exactor_model.py` can be imported or run directly to classify any sample.

```python
from exactor_model import MicroMindClassifier
import pandas as pd

clf = MicroMindClassifier()

# Example sample: [bottom-left, bottom-middle, bottom-right, middle-left, middle-middle, middle-right, top-left, top-middle, top-right]
X = pd.DataFrame([['x', 'x', 'o', 'o', 'o', 'x', 'x', 'o', 'x']], 
                 columns=['bottom-left-square', 'bottom-middle-square', 'bottom-right-square', 
                          'middle-left-square', 'middle-middle-square', 'middle-right-square', 
                          'top-left-square', 'top-middle-square', 'top-right-square'])

predictions = clf.predict(X)
print(predictions)
```

The script can also re-run the entire prediction set on the OpenML dataset when executed directly as `python exactor_model.py`.
