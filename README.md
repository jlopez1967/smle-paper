## What is this?

This repository contains all materials to **independently replicate and validate** the results from the paper:

> **Symbolic Machine Learning via Exhaustive Boolean Rule Induction: A New Paradigm Beyond Continuous Algebra**  
> J. LГіpez вЂ” MicroMind AI Project, March 2026

SMLE is a machine learning framework that operates entirely in **Boolean algebra** instead of continuous calculus. Instead of neural network weights, it learns human-readable logical rules like:

```
[top-left == x] AND [top-middle == x] AND [top-right == x]
в†’ X wins with 100% precision (8 instances)
```

## What SMLE is NOT

To avoid misunderstanding:
- вќЊ Not a general-purpose ML library (yet)
- вќЊ Not tested beyond Tic-Tac-Toe in this release (German Credit and Pima Diabetes are planned)
- вќЊ Not claiming to outperform neural networks on all tasks

## What SMLE IS

- вњ… A novel philosophical framework for Boolean-space learning
- вњ… A working implementation with reproducible results
- вњ… Fully interpretable by design вЂ” every prediction has a readable rule
- вњ… Compliant with EU AI Act explainability requirements

## How to Validate

1. **Verify metrics:** `python scripts/verify_results.py`
2. **Inspect the labeled data:** Open `data/dataset_etiquetado.csv` вЂ” every row has its assigned segment
3. **Read the rules:** The `Microsegmento_Descubierto` column tells you which logical rule each instance matched
4. **Spot-check manually:** Pick any row, find the corresponding segment, verify the board state satisfies the rule
